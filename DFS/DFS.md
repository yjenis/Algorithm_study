# DFS
DFS는 그래프의 시작 노드에서 출발하여 탐색할 한 쪽 분기를 정하여 최대 깊이까지 탐색을 마친 후 다른 쪽 분기로 이동하여 다시 탐색을 수행하는 알고리즘

#### 특징
1. 재귀 함수로 구현
2. 스택 자료구조 이용

#### 시간 복잡도
노드 수:V, 에지 수:E
O(V+E)

#### 응용하여 풀 수 있는 문제
단절점 찾기, 단절선 찾기, 사이클 찾기, 위상 정렬
<hr/>

### 핵심이론
- 한 번 방문한 노드를 다시 방문하면 안됨! -> 체크 리스트 필요
- 그래프는 **인접리스트**로 표현
- 탐색 방식은 **후입선출**이므로 **스택** 활용

##### 1. 초기 작업
- 인접 리스트로 그래프 표현
- 방문 리스트 초기화
- 시작 노드 스택에 삽입

##### 2. 스택에서 노드를 꺼낸 후 인접 노드 다시 삽입

##### 3. 스택에 자료구조에 값이 없을 때까지 반복
- 이 때 이미 다녀간 노드는 방문 리스트를 바탕으로 **재삽입X**

##### 추가
- 스택에 노드 **삽입**시 방문 리스트 체크
- 스택에서 노드 **제거**시 탐색 순서에 기록하며 인접 노드를 방문리스트와 대조하여 확인
- 삽입은 append() 꺼내는 것은 pop()
